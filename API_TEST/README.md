#### Run Migration :
- Linux
    ```bash
    NEST_ENV=[dev|test] npm run migrate:refresh  
    ```
- Windows
    ```bash
    SET NEST_ENV=[dev|test&&npm run migrate:refresh  
    ```

#### Develop Mode
- Linux
    ```bash
    npm run start:dev
    ```
- Windows
    ```bash
    npm run start:win-dev
    ```
#### Test Mode (Jest)
- Linux
    ```bash
    npm run start:test
    ```
- Windows
    ```bash
    npm run start:win-test
    ```
